# 《寒霜纪元：重生领主》互动影视游戏平台原型

这是根据 PRD v1.0 重做的互动影游网站工具。它不是营销落地页，而是可直接体验的前端产品原型。

## 打开方式

- 本地服务预览：`http://localhost:5178/`
- 静态文件入口：`index.html`

## 已实现

- 第一人称互动剧情播放界面
- 已接入 12 场景完整分支剧本：开场、2A/2B、3A/3B、4A/4B、5A/5B、6A/6B、四结局判定
- 剧情素材图片占位帧：重生醒来、超市封仓、北境庇护所
- “伪视频段落”播放：图片镜头推进 + 浏览器中文语音旁白 + 播放中隐藏选项
- 旁白与台词支持打字机动态显示
- 选项出现时以暂停态展示，后续可将节点 `media` 字段替换为 HLS/MP4 视频素材
- 选择驱动的剧情节点跳转
- 顶部基础属性条：资源、人口、武力、声望、寒意值
- 完整属性板折叠到角色中心，点击右上按钮展开
- NPC 关系网：林薇、赵擎、周砚
- 道具、成就、多结局记录
- 结局判定：寒霜帝王、暗影领主、商业之王、孤狼幸存者
- 会员隐藏支线与会员权益模拟
- 庇护所经营：仓储区、医疗站、武装岗哨、居住舱
- 生存任务与一键调度
- 剧情树节点编辑器和 JSON 导出
- 商城、会员、运营数据、REST API 草案面板
- 存档、读档、重置

## 数据说明

当前 Demo 使用浏览器 `localStorage` 模拟 `story_nodes`、`story_choices`、`game_sessions`、`player_stats`、`save_slots`、`items`、`achievements`、`vip_memberships` 等后端数据表。
